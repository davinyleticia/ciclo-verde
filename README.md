## ⚙️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone git@github.com:davinyleticia/ciclo-verde.git
```

### 2️⃣ Acessar a pasta do projeto

```bash
cd ciclo-verde
```

### 3️⃣ Abrir o projeto

Abra o arquivo `index.html` diretamente no navegador.
